아래 설명된 의미에 해당하는 용어를 쓰세요 
 반복문으로 for( A; B; C ){ D }로 구성되며, 초기화 A나 증감식 C를 수행하고, 조건식 B를 확인 후 참이면 명령문 D를 반복적으로 수행하고 거짓이면 반복문 중단하고 빠져나옴. 참고로, D는 0번 이상 실행  

 증가연산자로 a = a + 1를 의미하며, 변수 a에 저장된 값에 1만큼 증가시켜 변수 공간 a에 저장  

 비교연산자로 A가 B보다 작거나 같으면 참(true)을 반환하고 다르면 거짓(false)을 반환  

 print formatted의 줄임말로, 전달받은 내용을 형식에 맞게 모니터에 출력하는 메서드
