    입출력 개념

    프로그램 시작시 표준 입출력 스트림이 자동으로 생성되고, 프로그램 종료시 자동으로 소멸
    키보드     표준 입력 스트림
    System.in    		     프로그램     표준 출력 스트림
    System.out    		     모니터

    입력 프로그램 예제

    import java.util.Scanner;
    public class Main {
        public static void main( String[] args ) {
            System.out.print( "이름을 입력하세요: " );
            Scanner scan = new Scanner( System.in );
            String name = scan.next();
            System.out.println( "안녕하세요, " + name + "님. 만나서 반갑습니다" );
        }
    }

    import   다른 패키지에서 클래스를 가져오기

    java.util.Scanner   데이터를 읽어오는 기능을 제공하는 클래스의 패키지

    Scanner   다양한 종류의 데이터를 읽어오는 기능을 제공하는 클래스

    new   클래스 객체를 메모리에 생성

    System.in   키보드와 같은 표준 입력 스트림

    System.out   모니터와 같은 표준 출력 스트림

    next   문자열 데이터를 한 단어씩 읽어오는 메서드

    nextLine   문자열 데이터를 한 줄씩 읽어오는 메서드

    charAt   문자열에서 괄호안의 인덱스번호에 해당하는 문자 가져오기

    nextInt   정수형 데이터를 읽어오는 메서드

    nextDouble   실수형 데이터를 읽어오는 메서드

    입출력 속도 개선 프로그램

    import java.io.*;
    public class Main {
        public static void main( String[] args )  throws IOException  {
            System.out.print( "이름을 입력하세요: " );
            BufferedReader br = new BufferedReader( new InputStreamReader( System.in ) );
            BufferedWriter bw = new BufferedWriter(new OutputStreamWriter( System.out ) );
            String name = br.readLine();
            bw.write( "안녕하세요, " + name + "님. 만나서 반갑습니다" );
            bw.flush();
            bw.close();
            br.close();
        }
    }

    java.io.*   입출력과 관련된 클래스의 패키지

    throws   입출력 예외상황이 발생하면 예외처리 클래스로 보내는 명령어

    IOException   입출력시 발생할 수 있는 예외상황을 처리하기 위한 클래스

    BufferedReader   키보드와 CPU의 속도차를 고려하여, 키보드 데이터를 버퍼에 모았다가 한꺼번에 읽어오는 클래스

    BufferedWriter   모니터와 CPU의 속도차를 고려하여, 데이터를 버퍼에 모았다가 한꺼번에 모니터에 출력하는 클래스

    InputStreamReader   입력 스트림 데이터를 읽어오는 기능을 제공하는 클래스

    OutputStreamWriter   출력 스트림 데이터를 읽어오는 기능을 제공하는 클래스

    readLine   문자열 데이터를 한 줄씩 읽어오는 메서드

    write   주어진 내용을 출력하는 메서드

    flush   입출력 스트림을 비우는 함수

    close   입출력 스트림을 닫는 함수
