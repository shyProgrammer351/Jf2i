아래 설명된 의미에 해당하는 용어를 쓰세요 
 반복문으로 for( A; B; C ){ D }로 구성되며, 초기화 A나 증감식 C를 수행하고, 조건식 B를 확인 후 참이면 명령문 D를 반복적으로 수행하고 거짓이면 반복문 중단하고 빠져나옴. 참고로, D는 0번 이상 실행  

 증가연산자로 a = a + 1를 의미하며, 변수 a에 저장된 값에 1만큼 증가시켜 변수 공간 a에 저장  

 감소연산자로 a = a - 1를 의미하며, 변수 a에 저장된 값에 -1만큼 감소시켜 변수 공간 a에 저장  

 score[ index ]는 이전 값을 미리 temp에 저장한 후, 맞바꿀 값(score[ index + 1 ])으로 변경  

 score[ index + 1 ]는 이전 값을 미리 score[ index ]에 저장한 후, 맞바꿀 값(temp=score[ index ]의 초기값)으로 변경
